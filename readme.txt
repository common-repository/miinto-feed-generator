Miinto Feed Generator

== Description ==
Miinto Feed Generator by [The Right Software](https://therightsw.com/) web development company is a WooCommerce Plugin used to download products information in TSV format for Miinto feeds. This is only for Simple products.
Miinto is a fashion network created for all the fashion-forward consumers who demands the same high level service and experience as when they are shopping at their local shops offline. By providing a Product feed to Miinto, the plugin ensure automatic synchronization of your products to the Miinto Platform.

*Features:*

Create TSV file of Woocommerce Product Feeds for Miinto.

*Future Updates:*

It will work for variable and composite Woocommerce products.

== Installation ==
1. Upload (mfp-miinto-feeds)folder to the `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
